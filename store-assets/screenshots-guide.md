# Screenshots Guide

## Required sizes

| Store | Min size | Max | Count |
|---|---|---|---|
| Chrome Web Store | 1280×800 or 640×400 | — | 1–5 |
| Firefox AMO | 600px wide min | — | 1–10 |

## Shots to take (in order of priority)

1. **Button on Claude.ai** — floating ⊕ memory button visible, green dot, chat open
2. **Scope picker open** — dropdown showing scopes (global, project...) with counts
3. **Context injected** — chat input filled with memory context after clicking a scope
4. **Popup** — extension popup open showing "connected", scope name, memory count
5. **ChatGPT or Gemini** — same button on a different site (shows cross-platform support)

## Tips

- Use a dark theme on the AI site (matches the extension's dark UI)
- Have a few realistic memories in your store before screenshotting
- Use a 1280×800 browser window for clean captures
- Crop to the relevant area — don't show the full desktop
